# Asynchronous Logger Library

Welcome to the documentation for the Asynchronous Logger, a lightweight, high-performance logging solution for C++ and Python.

## Quick Overview

The logger is designed to provide a non-blocking logging experience. By offloading I/O operations to a background thread, your application can continue its main processing without being slowed down by terminal or file writes.

### Key Benefits
- **Pure C++11**: No heavy external dependencies for the C++ library.
- **Pythonic**: Easy-to-use Python implementation.
- **Stream-based**: Familiar C++ streaming syntax.
- **Async-First**: Never let logging block your main thread again.
